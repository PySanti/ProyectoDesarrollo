# Glossary

| Term | Meaning |
|---|---|
| UMBRAL | Platform for real-time immersive investigation experiences. |
| Trivia | Game mode based on quizzes, questions and answers. |
| Búsqueda del Tesoro | Game mode based on missions, stages, clues and evidence. |
| Session | Live execution of a game mode. |
| Team | Group of participants. |
| Evidence | Submission made by a team during Treasure Hunt. |
| Ranking | Ordered leaderboard based on team scores. |
| Audit event | Historical record of a relevant system action. |
| Integration event | Event published between microservices through RabbitMQ. |
