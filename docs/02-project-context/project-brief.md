# UMBRAL - Project Brief

UMBRAL is a web platform for operating real-time immersive investigation experiences.

The platform allows administrators and operators to prepare and execute live sessions where teams participate under one of two supported game modes:

- Trivia
- Búsqueda del Tesoro

The system replaces manual coordination through spreadsheets and messaging groups with a centralized, traceable, real-time platform.

## Main capabilities

- Manage users and roles.
- Manage teams and participants.
- Create live sessions.
- Operate Trivia sessions.
- Operate Treasure Hunt sessions.
- Register evidence and answers.
- Calculate scores and rankings.
- Publish real-time updates.
- Record audit history.
- Process asynchronous events.
