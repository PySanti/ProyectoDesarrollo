# Deprecated SDD Specs

This folder contains old or superseded SDD specs.

OpenCode must not use these specs for planning, design, implementation, review or traceability.

A deprecated spec can only be reactivated by regenerating it from:

- `docs/01-project-source/`
- `docs/02-project-context/`
- `docs/03-microservices/`
- `docs/04-sdd/SPECS-LIST.md`
- `contracts/`

and confirming it matches the current four-service topology.
